Server Rules

Read and follow these rules to keep the server fun and fair for everyone.

#01

No Griefing

Do not destroy, steal from, or damage other players' builds without explicit permission. This includes breaking blocks in someone else's base or farm.

#02

No Cheating or Hacks

No hacked clients, X-ray, or any cheat modifications. Only vanilla Minecraft or approved resource packs are allowed.

#03

Respect All Players

No harassment, hate speech, or toxic behavior in chat. Keep it friendly. This is a class server — act accordingly.

#04

PvP By Consent Only

Do not kill other players unless both parties agree to PvP. Check the Bounty Board — active bounties are the exception.

#05

Keep the Environment Clean

Don't leave floating trees, 1x1 towers, giant holes, or abandoned constructions. Clean up after mining and building.

#06

No Exploiting Bugs

If you find a dupe glitch or game-breaking bug, report it — don't use it. Fair economy is everyone's responsibility.

#07

Share Public Farms

If you build a public farm (mob farm, iron farm, etc.), let others know and don't lock them out without reason.

#08

Coordinate Before Big Projects

Planning a massive build? Post it in the Events section or tell the group first so nobody builds on your reserved land.
